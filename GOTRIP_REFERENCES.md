## Technology Stack References

  - https://www.w3.org/TR/html52/
  - https://www.w3.org/TR/CSS/

- **Vanilla JavaScript** - ECMAScript 2021 (ES12) Standard
  - https://tc39.es/ecma262/
  - MDN Web Docs: https://developer.mozilla.org/en-US/docs/Web/JavaScript

- **Chart.js** - Simple yet flexible JavaScript charting
  - https://www.chartjs.org/
  - Documentation: https://www.chartjs.org/docs/latest/

- **Bootstrap Icons** - Open source icon library
  - https://icons.getbootstrap.com/
  - GitHub: https://github.com/twbs/icons

### Backend Development
- **Node.js** - JavaScript runtime environment
  - Official: https://nodejs.org/
  - Documentation: https://nodejs.org/en/docs/

- **Express.js** - Minimalist web framework for Node.js
  - Official: https://expressjs.com/
  - Documentation: https://expressjs.com/en/api.html

- **bcryptjs** - Password hashing library
  - GitHub: https://github.com/dcodeIO/bcrypt.js
  - NPM: https://www.npmjs.com/package/bcryptjs

- **JWT (JSON Web Tokens)** - Authentication standard
  - Official: https://jwt.io/
  - RFC 7519: https://tools.ietf.org/html/rfc7519

### Database
- **MySQL** - Relational database management system
  - Official: https://www.mysql.com/
  - Documentation: https://dev.mysql.com/doc/

- **SQL** - Structured Query Language
  - W3Schools SQL Tutorial: https://www.w3schools.com/sql/
  - SQL Standard: https://www.iso.org/standard/63555.html
